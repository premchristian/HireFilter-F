---------------------------------------------------------------------------------------
1️⃣ Stage your changes
git add .


. adds all changed/new files. You can also add a specific file like:

git add filename.js

2️⃣ Commit your changes
git commit -m "Describe what you changed"


Example:

git commit -m "Added login page functionality"

3️⃣ Push changes to GitHub
git push


This will upload your latest commit to the GitHub repository.

✅ Summary for every update:
